# Taist.Framework
## Introduction
*Taist.Framework* is an automated testing framework which record the tracks of tester's behavior.<br>
it has the following characteristics:
- Simpleness
- Flexible
- High reuse
- Modularization
- Good expansibility
- Data driven
- Keyword driven

## Support test
- Windows UI
- InternetExplore UI
- ActiveX Interface
- Http(s) Interface
- Soap Webservice Interface
